# Creative Project Planning

## When Chris Starts a Creative Project
Create a project with category=creative. Use meta fields for format-specific tracking.

## Supported Formats
- **Book**: Track word count target/current. Sections as issues.
- **Game**: Design docs as issues, mechanics as tasks.
- **Show**: Episodes/scenes as issues, rehearsal schedule via calendar.
- **Course**: Modules as issues, lessons as tasks.

## Meta Fields
- format: book, game, show, course, other
- word_count_target: for writing projects
- word_count_current: updated as progress happens

## Creative Project Principles
- These are usually speculative — no money attached initially
- Don't let them crowd out paid work (see TRIAGE.md)
- Break into small deliverables to maintain momentum
- Use calendar events for dedicated creative time blocks
- Track progress concretely (word count, scenes written, etc.)
