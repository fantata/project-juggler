# Podcast Planning

## When Chris Wants to Plan a Podcast
Create a project with category=podcast. Episodes are "issues" with meta fields.

## Episode Workflow
1. Idea/topic → create issue with status=open, meta.status=pitched
2. Guest identified → update meta with guest_name, guest_email, guest_bio
3. Guest confirmed → meta.status=confirmed, create calendar event for recording
4. Recorded → meta.status=recorded, create editing tasks
5. Edited → meta.status=edited, schedule publish date
6. Published → meta.status=published, issue status=done

## Segment Structure (use as task template)
When creating an episode, auto-suggest these tasks:
- Intro/cold open
- Guest introduction
- Main topic discussion
- Segment: [topic-specific game or bit]
- Wrap-up and plugs
- Post-production: edit audio
- Post-production: write show notes
- Post-production: create social clips
- Publish and distribute

## Guest Outreach
When drafting guest outreach, Chris's tone is casual and warm — he's an
improviser inviting someone to have a fun conversation, not a formal
interview request. Keep it short. Mention the show format and why he thinks
they'd be great.
